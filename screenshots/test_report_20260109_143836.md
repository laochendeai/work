## 技术实现

### Radio 菜单

```python
page = st.radio(
    "📋 功能菜单",
    [
        "📊 系统概览",
        "🕷️ 数据爬取",
        "👥 联系人管理",
        "📧 邮件营销",
        "📥 数据导出",
        "🗄️ 数据库管理",
        "📋 系统日志",
        "⚙️ 系统设置",
    ],
    index=0,
    horizontal=False,
    label_visibility="visible"
)
```

**优势**:
- 所有 8 个选项默认可见
- 无需点击即可看到所有功能
- 更好的用户体验
