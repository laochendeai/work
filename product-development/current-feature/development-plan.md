## 3. 技术架构（个人版）

### 3.1 技术选型
- **开发语言**: Python 3.8+（免费开源）
- **爬虫框架**: BeautifulSoup4 + requests（轻量级）
- **数据库**: SQLite（本地文件，无需服务器）
- **邮件服务**: SMTP（个人邮箱，免费）
- **桌面界面**: Tkinter（Python自带）
- **定时任务**: schedule库（免费）
- **开发工具**: VS Code（免费）

### 3.2 系统架构
```
┌─────────────────┐
│   桌面界面      │ ← Tkinter GUI
├─────────────────┤
│   控制器        │ ← 主程序逻辑
├─────────────────┤
│   数据采集层    │ ← 爬虫模块
├─────────────────┤
│   信息处理层    │ ← 提取与验证
├─────────────────┤
│   邮件发送层    │ ← SMTP模块
├─────────────────┤
│   数据存储层    │ ← SQLite数据库
└─────────────────┘
```

### 3.3 模块设计
- **main.py**: 主程序入口和GUI界面
- **scraper.py**: 爬虫模块，负责网站数据采集
- **parser.py**: 信息解析模块，提取联系人信息
- **emailer.py**: 邮件发送模块，自动化营销
- **database.py**: 数据库操作模块，SQLite管理
- **scheduler.py**: 定时任务模块，自动化调度
- **config.py**: 配置管理模块，个人信息设置
