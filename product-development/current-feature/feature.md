# 个人智能设计营销自动化系统

## 功能概述

个人智能设计营销自动化系统是专为智能化设计工程师打造的自动化营销工具，能够7*24小时自动搜索政府采购、高校、国企、上市公司的智能化相关中标公告，提取联系人信息，并通过自动化方式向潜在客户推荐个人设计团队，帮助获得更多招投标方案设计和施工方案设计项目机会。

## 核心功能

### 1. 全渠道智能化项目搜索
- **政府采购**: 中国政府采购网(ccgp.gov.cn)及重点省市采购网
- **高校采购**: 985/211高校采购平台（清华、北大、复旦等）
- **国企央企**: 中石油、国家电网、中国移动等央企采购平台
- **大型企业**: 华为、腾讯、阿里、万科等上市公司采购平台
- **专业领域**: 教育装备、医疗设备、交通等行业采购平台

### 2. 智能联系信息提取
- **联系人识别**: 自动识别项目经理、负责人、联系人信息
- **联系方式提取**: 智能提取手机号、座机、邮箱地址
- **信息验证**: 邮箱格式验证和电话号码有效性检查
- **数据清洗**: 自动去重和格式标准化

### 3. 个性化邮件营销
- **模板管理**: 可自定义的邮件模板和个人介绍
- **智能发送**: 根据项目特点生成个性化推荐邮件
- **发送控制**: 智能控制发送频率，避免垃圾邮件标记
- **效果跟踪**: 邮件送达率、打开率、回复率统计

### 4. 客户数据库管理
- **本地存储**: SQLite数据库，安全可靠
- **分类管理**: 按政府、高校、国企、企业分类存储
- **跟进记录**: 记录发送历史和客户回复情况
- **数据导出**: 支持Excel格式的客户信息导出

- 我想要快速获得项目联系人的电话和邮箱，直接进行业务推广
- 我需要自动化的营销工具，节省时间专注于设计工作本身

### 作为自由职业的设计专家
- 我希望建立持续的项目获取渠道，不再依赖人工寻找客户
- 我想要自动向潜在客户展示我的专业能力和项目经验
- 我需要管理不同来源的客户信息，便于长期跟进

## 技术要求（个人版）

### 数据采集要求
- **轻量级架构**: Python + BeautifulSoup4，单机运行
- **智能反爬**: 合理频率控制，模拟正常用户行为
- **采集频率**: 每小时检查一次，避免过度请求
- **数据准确性**: 联系人信息提取准确率>70%

### 智能分析要求
- **关键词匹配**: "智能化"、"弱电"、"安防"、"监控"等
- **项目筛选**: 重点关注10万-1000万的智能化项目
- **信息提取**: 姓名、电话、邮箱至少有一项有效信息
- **分类准确性**: 政府40%、高校20%、国企25%、企业15%

### 性能要求（个人版）
- **硬件要求**: 普通家用电脑，4GB内存以上
- **运行要求**: 7*24小时稳定运行，异常自动重启
- **响应时间**: 本地查询<3秒，邮件发送及时
- **存储容量**: SQLite本地数据库，支持TB级扩展

### 成本控制要求
- **开发成本**: 0元（使用免费工具）
- **运行成本**: 0元（个人电脑+免费邮箱）
- **维护成本**: 最小化，代码简单易维护
- **总预算**: 500元以内
